# نظام تسجيل الدخول

مشروع ويب بسيط يتضمن صفحات تسجيل الدخول وإنشاء حساب جديد والصفحة الرئيسية.

## المميزات

- صفحة تسجيل الدخول
- صفحة إنشاء حساب جديد
- الصفحة الرئيسية
- تصميم متجاوب
- واجهة مستخدم بسيطة وسهلة الاستخدام

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
